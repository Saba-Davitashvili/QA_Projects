# [BUG-031] Chatbot Widget Icon Overlaps Profile Navigation Button on Mobile Viewports

## Metadata
- **Project**: On-Demand Gardening & Landscaping Service Platform
- **Component**: UI / Responsive Layout
- **Type**: Bug / UI
- **Severity**: Critical
- **Priority**: Critical
- **Status**: Open

## Environment Details
- **Platform**: Mobile (iPhone, iOS 26.5, Safari)
- **App Context**: Bottom Navigation Bar

## Preconditions
User is logged in to the application from a mobile device.

## Steps to Reproduce
1. Observe the bottom navigation bar and chatbot widget position.

## Actual Result
The chatbot floating icon covers the "Profile" navigation button, blocking access to it.

## Expected Result
The floating chatbot icon should be offset or have a margin that prevents it from overlapping the navigation bar buttons.
