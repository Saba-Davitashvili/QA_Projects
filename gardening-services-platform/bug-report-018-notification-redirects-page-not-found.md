# [BUG-056] Clicking Service Request Notification Redirects Gardener to Page Not Found Screen

## Metadata
- **Project**: On-Demand Gardening & Landscaping Service Platform
- **Component**: Gardener Portal / Navigation
- **Type**: Bug / Functional
- **Severity**: Critical
- **Priority**: Critical
- **Status**: Open

## Environment Details
- **Platform**: Desktop / Laptop (Windows 11, macOS 14)
- **App Context**: Notifications Section

## Preconditions
User profile is verified as a gardener.
The gardener has received a request notification from a client.

## Steps to Reproduce
1. Log in to the application.
2. Open the notifications section.
3. Click on the service request notification.

## Actual Result
The gardener is redirected to a "Page Not Found" error screen.

## Expected Result
The gardener should be navigated to the corresponding service request details page.
