# [BUG-023] Gardener Profile Button Leads to Page Not Found Error Page

## Metadata
- **Project**: On-Demand Gardening & Landscaping Service Platform
- **Component**: Gardener Profile / Navigation
- **Type**: Bug / Logic
- **Severity**: Critical
- **Priority**: Critical
- **Status**: Open

## Environment Details
- **Platform**: Desktop / Laptop (Windows 11, macOS 14)
- **App Context**: Gardener Selection Screen

## Preconditions
User is logged in and has successfully submitted a service request.

## Steps to Reproduce
1. Navigate to the matching gardeners list.
2. Click the "Profile" button on any gardener card.

## Actual Result
The user is redirected to a "Page Not Found" error page.

## Expected Result
The button should navigate to the gardener's profile details page showing their specific ratings and service history.
